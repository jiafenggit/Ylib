yv_new

yvect_t yv_new(size_t size);
Creates a new yvector of the given size. Use the default size (4 KB) if the given size is zero.

Parameters
size Size of the new yvector.

Result: The created yvector.

yv_free

void yv_free(yvect_t *v, void (*f)(void*, void*), void *data);
Delete an existing yvector.

Parameters
v A pointer to the yvector.
f Pointer to a function that is called to delete each element of the vector. Could be NULL ; otherwise, must have this prototype : void f(void *elem, void *data);
data Pointer to data that could be given to the delete callback (see previous parameter). Could be NULL.

yv_trunc

void yv_trunc(yvect_t v, void (*f)(void*, void*), void *data);
Truncate an existing yvector. The allocated memory doesn't change.

Parameters
v The yvector.
f Pointer to a function that is called to delete each element of the vector. Could be NULL ; otherwise, must have this prototype : void f(void *elem, void *data);
data Pointer to data that could be given to the delete callback (see previous parameter). Could be NULL.

yv_ins

int yv_ins(yvect_t *v, void *e, size_t i);
Insert an element at the given offset of a yvector. All elements placed at this offset and after are shifted to the right.

Parameters
v A pointer to the yvector.
e A pointer to the element.
i Offset of the element in the vector. Must be less or equal to the vector's used size.

Result: 0 if an error occurs, 1 otherwise.
