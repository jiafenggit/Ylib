ylog_s

struct ylog_s {
	ylog_priority_t threshold;
	char *progname;
	char *filename;
	void (*handler)(const char*);
	int setup;
	int facility;
};
Main structure for ylogs. Must have one of it in global space.

Fields
threshold Current minimum priority level of written logs.
progname Name of the current program.
filename Path to the output file.
handler Function pointer to the log handler (set to NULL for logs on stderr, syslog or file).
setup Log type (stderr, syslog, file).
facility Syslog facility.
