ylog_type_e

enum ylog_type_e {
	YLOG_STDERR	= 1,
	YLOG_FILE	= 2,
	YLOG_SYSLOG	= 4,
	YLOG_HANDLER	= 8
};
All different destinations possible for logs.

Constants
YLOG_STDERR Logs are written on the standard error output.
YLOG_FILE Logs are written on a given file.
YLOG_SYSLOG Logs are written on the system logger.
YLOG_HANDLER Logs are sent to a given handler.

ylog_priority_e

enum ylog_priority_e {
	YLOG_FORCE = -1,
	YLOG_DEBUG = 0,
	YLOG_INFO,
	YLOG_NOTE,
	YLOG_WARN,
	YLOG_ERR,
	YLOG_CRIT
};
Several priority levels are defined in ylogs. They are similar to the syslog's ones (see 'man 3 syslog')

Constants
YLOG_FORCE Force message writing (don't use it for threshold).
YLOG_DEBUG Debug-level message (lower priority level).
YLOG_INFO Informational message (default level).
YLOG_NOTE Notice ; normal but significant.
YLOG_WARN Warning.
YLOG_ERR Error.
YLOG_CRIT Critical message (higher priority level).
