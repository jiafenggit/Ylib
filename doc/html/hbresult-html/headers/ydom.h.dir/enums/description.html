ydom_node_type_e

enum ydom_node_type_e {
	VOID_NODE = 0,
	ELEMENT_NODE,
	ATTRIBUTE_NODE,
	TEXT_NODE,
	CDATA_SECTION_NODE,
	PROCESSING_INSTRUCTION_NODE,
	COMMENT_NODE,
	DOCUMENT_NODE
};
Node type definitions.

Constants
VOID_NODE Node without any type.
ELEMENT_NODE It is an element (node of the XML tre)e
ATTRIBUTE_NODE The node is an attribute of a node.
TEXT_NODE The node contains text only.
CDATA_SECTION_NODE The node correspond to a CDATA section.
PROCESSING_INSTRUCTION_NODE The node contains some processing instruction.
COMMENT_NODE XML comment node.
DOCUMENT_NODE Root node that represent the whole XML document.
