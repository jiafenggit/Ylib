Structures


ydom_s

struct ydom_s {
	struct ydom_node_s *document_element;
	struct ydom_node_s *current_parsed_node;
	yerr_t error;
	char *xml_version;
	char *encoding;
	char *standalone;
};
Structure for DOM representation.

Fields
document_element Pointer to the root document node.
current_parsed_node Pointer used during XML file parsing.
error Error status value.
xml_version String which contains the XML version of the document.
encoding String which contains the type of document encoding.
standalone "yes" if the document has a DTD.

ydom_node_s

struct ydom_node_s {
	ydom_node_type_t node_type;
	bool complete;
	char *name;
	char *value;
	int position;
	struct ydom_node_s *parent;
	struct ydom_node_s *prev;
	struct ydom_node_s *next;
	struct ydom_node_s *attributes;
	struct ydom_node_s *first_child;
	struct ydom_node_s *last_child;
	struct ydom_node_s *document;
};
Structure for XML Node representation.

Fields
node_type Type of the node.
complete Set to TRUE when the parse of the node in the XML file is complete.
name Name of the node.
value Value of the node.
position Node posision relative to its sibblings.
parent Pointer to the parent node.
prev Pointer to the previous sibling of the node.
next Pointer to the next sibling of the node.
attributes Pointer to the first attribute of the node.
first_child Pointer to the first child of the node.
last_child Pointer to the last child of the node.
document Pointer to the document node of this node.
