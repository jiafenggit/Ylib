ymap_t

typedef struct ymap_s {
	size_t		size;
	size_t		used;
	ymap_bucket_t	*buckets;
	ymap_function_t	destroy_func;
	void		*destroy_data;
} ymap_t;


Fields
size Current size of the hash map.
used Current number of elements stored in the hash map.
buckets Array of buckets.
destroy_func Pointer to the function called when an element is removed.
destroy_data Pointer to some user data given to the destroy function.
