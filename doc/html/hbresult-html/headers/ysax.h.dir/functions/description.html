ysax_new

ysax_t *ysax_new(void *parse_data);
Create a SAX XML parser.

Parameters
parse_data Pointer to data that can be used by handlers.

Result: A pointer to the created parser, or NULL if an error occurs.

ysax_read_handler

yerr_t ysax_read_handler(ysax_t *sax, char (*getc_hdlr)(void*),
			 void (*ungetc_hdlr)(char,void*), void *xml_data);
Launch the parsing of some XML data, using handlers to read data.

Parameters
sax A pointer to the SAX object.
getc_hdlr Handler called to get one character.
ungetc_hdlr Handler called to unget one character.
xml_data Pointer to data used by handlers.

Result: An error code that show the return status of parse action.

ysax_stop

void ysax_stop(ysax_t *sax);
Stop the SAX parser. Use this function if you detect an error in a handler. The parser will return an YEL2HLT error (Level 2 halted).

Parameters
sax A pointer to the Sax parser.

When a yvector of XML attributes is destroyed, this function can be passed as parameter to yv_del() to free all memory allocated for each element.

Parameters
