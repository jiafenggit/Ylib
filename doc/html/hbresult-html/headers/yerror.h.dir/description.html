yerror.h


Abstract: Definitions for error handling.

The type yerr_t is a convenient way to transmit error values. When a yerr_t value is positive, there is no error. If it is negative or equal to zero, there is an error. When a function have to return an error code, it must return YENOERR in case of success, or a code that depend of the occuring error.
